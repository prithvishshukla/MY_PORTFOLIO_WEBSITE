1. Clone the repository:
```bash
git clone https://github.com/prithvishshukla/MY_PORTFOLIO_WEBSITE.git
```

2. Navigate to the project directory:
```bash
cd MY_PORTFOLIO_WEBSITE
```

```bash

python -m http.server 8000
```
